# VAX: Data annotation tool (Streamlit) for custom real-time data collection using sensors 

This tool is used to annotate labels while collecting data from users using any kind of unix timestamped sensor data. This tool is built as an streamlit application.

For more details, contact [prasoonpatidar@cmu.edu](prasoonpatidar@cmu.edu).
